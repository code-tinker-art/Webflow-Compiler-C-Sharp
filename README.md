# Getting Started

## Prerequisites
- Ensure you have Visual Studio 2022 or Visual Studio Code installed.
- Install the .NET SDK (6.0) or higher: [Download .NET](https://dotnet.microsoft.com/download)

## Cloning the Repository
1. Open a terminal (Command Prompt, PowerShell, or Git Bash).
2. Clone the repository using the following command:
   ```bash
   git clone https://github.com/code-tinker-art/Webflow-Compiler-C-Sharp.git
   cd Webflow-Compiler-C-Sharp
   ```

## Build Instructions
### For Visual Studio 2022
1. Open the solution file (`.sln`) in Visual Studio.
2. Restore the NuGet packages if prompted.
3. Build the solution by navigating to `Build > Build Solution` in the menu.

### For .NET CLI
1. Open a terminal and navigate to the project directory.
2. Run the following command to build the project:
   ```bash
   dotnet build
   ```
